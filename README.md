# Aplikasi Invoice Management #

Aplikasi ini dipakai untuk mengelola invoice dengan berbagai metode pembayaran masa kini.
Diantara metode pembayaran yang akan di support di antara lain :

* Virtual Account Bank

    * Bank BNI
    * Bank CIMB
    * Bank BSI

* e-wallet
    * Ovo
    * Gopay

* QR Payment
    * QRIS
